# **App Name**: FrikiBox

## Core Features:

- Personalized Box Builder: Allows users to customize their FrikiBox with selected themes, Funko Pops, clothing, and accessories.
- Product Gallery: Displays examples of available items and pre-made FrikiBoxes with a carousel or responsive grid.
- Testimonial Section: Showcases customer reviews and testimonials to build trust and credibility.
- Order Management: Integrates with Firestore to store and manage customized order details.
- Contact Form: Enables users to send messages and inquiries, storing the submissions in Firestore.
- Social Media Links: Provides links to the company's social media profiles (Instagram, TikTok, WhatsApp).
- Themed Suggestion Tool: Suggests a theme to users based on their hobbies. Generative AI analyzes hobbies to select the best theme.

## Style Guidelines:

- Primary color: Vivid purple (#9400D3) to represent the fantastical nature of fandoms.
- Background color: Light grey (#E0E0E0) for a clean and modern backdrop that highlights the colorful products.
- Accent color: Electric blue (#7DF9FF) for call-to-action buttons and interactive elements to draw attention.
- Headline font: 'Belleza' (sans-serif) for headings and short text with a touch of elegance. Body text: 'Alegreya' (serif) for longer text, to improve readability.
- Use custom SVG icons related to video games, series, and comics, ensuring they are interactive and align with the theme.
- Implement a responsive grid layout for the product gallery and other sections to ensure compatibility across devices.
- Incorporate subtle CSS/JS animations (or Framer Motion for React) for a modern, dynamic user experience.
